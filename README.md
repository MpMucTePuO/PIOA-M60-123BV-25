## Структура проекта

### main.py

Точка входа. Запускает текстовый интерфейс программы.

### tui.py

Модуль с текстовым интерфейсом пользователя.

Отвечает за:

- вывод меню
- получение данных от пользователя
- вызов функций работы с базой данных
- вывод результатов на экран

#### Вспомогательные функции ввода:

- `_read_int()` — считывает целое число, повторяет запрос при ошибке
- `_read_optional_int()` — считывает целое число или возвращает `None` при пустом вводе
- `_read_float()` — считывает число с плавающей точкой, повторяет запрос при ошибке
- `_read_optional_float()` — считывает число с плавающей точкой или возвращает `None` при пустом вводе
- `_read_nonempty_string()` — считывает непустую строку, повторяет запрос при ошибке

#### Функции вывода:

- `_print_employees()` — выводит список сотрудников в виде таблицы

#### Функции работы с сотрудниками:

- `_add_employee()` — диалог добавления сотрудника
- `_find_employees()` — диалог поиска сотрудников по фильтрам
- `_update_employee()` — диалог обновления данных сотрудника
- `_delete_employee()` — диалог удаления сотрудника

#### Меню:

- `run()` — главная функция запуска, отображает основное меню

### backend/memory.py

Модуль хранения данных в оперативной памяти.

#### Типы

- `EmployeeRecord = tuple[int, str, str, str, float]` — запись сотрудника: `(id, имя, должность, отдел, зарплата)`

#### Таблицы

- `Employees: list[EmployeeRecord]` — хранилище сотрудников

#### Функции для работы с сотрудниками

- `create_employee(employee_id, name, position, department, salary)` — создаёт сотрудника и добавляет в `Employees`
- `select_employees(employee_id, name, position, department, min_salary, max_salary)` — возвращает сотрудников по
  заданным фильтрам; без фильтров — всю таблицу
- `update_employee(employee_id, name, position, department, salary)` — обновляет данные сотрудника по `id`
- `delete_employee(employee_id)` — удаляет сотрудника по `id`

#### Вспомогательные функции

- `_employee_exists(employee_id)` — возвращает `True`, если сотрудник с указанным `id` существует

## Инструкция по запуску проекта
