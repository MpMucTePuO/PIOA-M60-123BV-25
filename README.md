# База данных «Сотрудники компании»

Учебный прототип базы данных, работающей в оперативной памяти.
Содержит одну таблицу `Employees` и простой консольный интерфейс.

## Структура проекта

```
company_db/
├── backend/
│   ├── __init__.py
│   └── memory.py      
├── __init__.py
├── __main__.py       
└── tui.py           
README.md
```

## Реализованная функциональность

- Добавление сотрудника с проверкой корректности данных (ID, имя, должность, зарплата).
- Чтение записей с фильтрацией по любому набору полей,
  включая диапазон зарплаты (`min_salary`, `max_salary`).
- Обновление записи по ID (любые поля, кроме ID).
- Удаление записи по ID.
- Вывод всех сотрудников.
- Обработка ошибок: некорректный ввод, дубликаты ID,
  обращение к несуществующим записям, пустые обязательные поля,
  отрицательная зарплата.

## Запуск
